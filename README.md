# Todo CLI

**No external dependancies! Just run it anywhere!**

## Getting started

Only Node.js required. Then, you can directly run,

`totodocli <command> <args>`

Usage :

* `ls  -          Show remaining todos`
* `del NUMBER  -  Delete a todo`
* `done NUMBER  - Complete a todo`
* `help  -        Show usage`
* `report  -      Statistics`
